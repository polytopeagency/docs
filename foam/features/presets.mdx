FOAM includes 99 factory presets organized by category and playing style.

## Categories

| Category | Character |
|----------|-----------|
| **Bass & Sub** | Deep, low-end focused bubbles |
| **Cinematic** | Atmospheric, evolving textures |
| **Extreme** | Aggressive, dense cascades |
| **Musical** | Melodic, tonal bubble sounds |
| **Nature** | Organic, realistic water textures |
| **Percussion** | Rhythmic, transient-focused |
| **Texture** | Ambient, layered soundscapes |
| **Tonal** | Pitched, harmonic content |

## Tier Tags

Each preset is tagged with a playing style indicating how it's designed to be used:

| Tier | Description |
|------|-------------|
| **Single** | Designed for single note playing |
| **Triad** | Works well with 3-note chords |
| **Layer** | Meant to be layered with other sounds |
| **Cluster** | Dense multi-note clusters |
| **Sequence** | Rhythmic or sequential patterns |

Preset names include their tier, e.g., "Eclipsed Singularity (Cluster)".

## Browsing Presets

Use the **Preset** dropdown in the header bar. Presets are organized with section headings by category and tier for easy browsing.

## Saving User Presets

The Preset dropdown includes an option to save your current settings as a user preset.

## Preset Persistence

Presets are fully saved with your DAW session, including all parameter values, MixCard settings (Drive and band knobs), and voice scaling configuration.
