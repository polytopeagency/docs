FOAM's interface is organized into five main cards:

| Card | Location | Purpose |
|------|----------|---------|
| **Bubble Physics** | Top Left | How bubbles emerge, pop, ring, and fade |
| **Fluid Dynamics & Acoustics** | Bottom Left | The liquid medium and physical resonator body |
| **Enhancers** | Top Right | Spatial effects, modulation, and temporal dynamics |
| **Pitch Gravity Logic** | Bottom Right | Scale quantization and ADP psychoacoustics |
| **Foam Voicing Dynamics** | Center Bottom | Voice scaling and parallel compression |

## Header Bar

The top bar contains:

- **Preset dropdown** — browse and save presets
- **MIDI Learn** button — click, then right-click any knob to assign a MIDI CC
- **CPU Cut** button — reduces density for low-power machines
- **GPU Cut** button — disables the visualizer to save GPU resources
- **OSC Setup** button — configure OSC output (green when active)

## Parameter Locking

Right-click any knob to **lock** it. Locked parameters are excluded from:
- Chaos randomization (both Jump and LFO modes)
- Drift modulation
- Preset changes (locked knobs retain their current value)

## Right-Click Toggles

A few controls have additional behavior on right-click:

- **Chaos knob** — toggles between Jump mode (per-note randomization) and LFO mode (continuous multi-rate modulation). Label changes to "Chaos LFO" in LFO mode.
- **Any knob** — locks/unlocks it from modulation as described above.

## License Status

The bottom credits bar shows your license status: **Licensed**, **Trial** (with days remaining), or **Unlicensed**. Click the status text to open the license dialog and enter your trial or full license key.
