## Custom Scale

Set the `customPitchClasses` parameter (0–4095) to define any combination of the 12 pitch classes. Each bit represents a pitch class: bit 0 = C, bit 1 = C#, ..., bit 11 = B. For example, 2741 (binary: 101010110101) = C Major.
