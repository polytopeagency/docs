# Installation

## System Requirements

- **OS:** macOS (Apple Silicon + Intel)
- **Formats:** VST3, Audio Unit (AU), Standalone
- **DAW:** Any VST3 or AU-compatible host (Ableton Live, Logic Pro, Reaper, Bitwig, etc.)

## Download

Download the latest PERF installer from [nomn.ai](https://nomn.ai).

## Install

Run the installer. It places plugin files in the standard locations:

| Format | Location |
|--------|----------|
| **VST3** | `~/Library/Audio/Plug-Ins/VST3/PERF.vst3` |
| **AU** | `~/Library/Audio/Plug-Ins/Components/PERF.component` |
| **Standalone** | `/Applications/PERF.app` |

## DAW Setup

### Ableton Live
1. Open **Preferences → Plug-Ins**
2. Enable **VST3** and set the custom folder if needed
3. Rescan plugins
4. Find PERF under **Plug-Ins → NOMN**

### Logic Pro
1. Open **Logic Pro → Settings → Audio Units**
2. Logic scans AU plugins automatically on launch
3. Find PERF under **MIDI FX → NOMN → PERF**

### Reaper
1. Open **Preferences → Plug-Ins → VST**
2. Add VST3 path if not already listed
3. Rescan
4. Insert PERF on any track via **FX → VST3**

## MIDI Routing

PERF generates MIDI output. To hear sound:

1. Load PERF on a **MIDI track** or **instrument track**
2. Route PERF's MIDI output to a virtual instrument (synth, sampler, etc.)
3. Press play — PERF generates MIDI in sync with your DAW transport

<Tip>
In standalone mode, PERF uses an internal clock. Press the **Generation** toggle to start producing MIDI, then route output to your system's virtual MIDI bus.
</Tip>

## Uninstall

Delete the plugin files from the locations listed above. User presets are stored in `~/Library/Application Support/PERF/` — remove this folder to clear all saved data.
